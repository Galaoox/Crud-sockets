# Crud-sockets

Application of notes that allow create, update, get and delete notes with socket.io

## Tech Stack

**Client:** Vue.js, HTML, CSS

**Server:** Node, Express, Socket.io

## Installation

Install Crud-sockets with yarn

```bash
  cd Crud-sockets
  yarn install
```
